SEGUNDA PAGINA

En la segunda pagina utilize una hojas de estilo externas se consigue separar el archivo de estilos del fichero HTML. . Este es el método más eficiente y más sencillo de mantener ya que el código CSS se encuentra separado del fichero HTML.

TERCERA PAGINA

En la tercera pagina utilice la forma directa o , estilos en líneaL son declaraciones CSS que se integran en las etiquetas HTML mediante el atributo style. Este método tan solo afecta al elemento en el que se integra el código.
